# Tutorials

- `single_sed_scipy.py`: fast deterministic fit.
- `single_sed_mcmc.py`: MCMC initialization, backend storage, and restart.
- `additive_sed_mcmc.py`: multiple rings or grain compositions.
- `single_sed_nested.py`: nested sampling.

Each file is a configuration template: replace the observation arrays and
material/star names with the values for your target.
